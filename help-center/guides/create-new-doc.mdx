## Add a New Doc

<Steps>
  <Step title="Go to Dashboard">
    Log in at `https://ponomarev-docs.com/dashboard`.
  </Step>
  <Step title="Click 'New Page'">
    Tap the 'New Page' button to start a fresh document.
  </Step>
  <Step title="Fill in Details">
    - Add a title
    - Write page content in the editor
  </Step>
  <Step title="Save and Publish">
    Click 'Save' to draft, or 'Publish' to go live.
  </Step>
</Steps>
